# Test di Carrello E-Commerce con Stripe

Questo è un progetto di esempio che implementa un carrello e-commerce con pagamento tramite **Stripe**. Il sistema consente di aggiungere prodotti al carrello e di effettuare il pagamento utilizzando il servizio **Stripe Checkout**.

## Funzionalità
- Aggiungi prodotti al carrello.
- Visualizza il totale dell'ordine.
- Effettua il pagamento tramite Stripe Checkout.

## Prerequisiti

Prima di eseguire il progetto, assicurati di avere i seguenti requisiti:
- PHP >= 7.x
- Composer (per gestire le dipendenze)
- Un account **Stripe** per configurare i pagamenti
